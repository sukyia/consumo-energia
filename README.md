# Programa em Python que calcula o consumo mensal de um aparelho.

Como funciona:
- Nome do aparelho
- Potência (W)
- Horas por dia

O programa calcula o consumo em kWh.

Fórmula:
(consumo) = (potência * horas * 30) / 1000

Como usar:
No terminal:
python app.py

Exemplo:
Aparelho: TV  
Consumo mensal: 90 kWh
