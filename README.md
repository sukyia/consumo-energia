#**Nome e objetivo:** Calculadora de Consumo de Energia – calcula o consumo mensal de um aparelho elétrico.

**Linguagem usada:** Python

**Fórmula utilizada:**  
(consumo) = (potência * horas por dia * 30) / 1000

**Como executar:**  
1. Abra o terminal  
2. Digite: `python app.py`  
3. Informe o nome do aparelho, potência e horas de uso por dia  

**Exemplo:**  
Aparelho: TV  
Consumo mensal: 90 kWh
